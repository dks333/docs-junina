# Source: https://mintlify.com/docs/editor/live-preview

Preview your site while you edit. Live previews render pages exactly as they look when published, and update as you make changes to content, navigation, and configuration.

## 

[​](https://mintlify.com/#open-live-preview)

Open live preview

Click the live preview button in the top bar to open a live preview in a new tab.

![The live preview button in the toolbar.](https://mintcdn.com/mintlify/i7HF0rh5FCMSySpX/images/editor/live-preview-light.png?fit=max&auto=format&n=i7HF0rh5FCMSySpX&q=85&s=58df3e4777909066be3e480ec4ed3daf)![The live preview button in the toolbar.](https://mintcdn.com/mintlify/i7HF0rh5FCMSySpX/images/editor/live-preview-dark.png?fit=max&auto=format&n=i7HF0rh5FCMSySpX&q=85&s=afea2c09afff0e761cb65aa5db39149f)

Live preview is only available for public sites.

## 

[​](https://mintlify.com/#live-preview-versus-preview-deployments)

Live preview versus preview deployments

| | Live preview | Preview deployment |
| --- | --- | --- |
| **Speed** | Instant | Requires a build |
| **Access** | Local URL | Shareable URL |
| **Use when** | Reviewing changes while editing | Sharing with your team for review |
| **Availability** | While the editor is open | As long as the branch exists |

Use live preview for immediate feedback while editing. When you need to share changes with your team or test on a different device, use a [preview deployment](https://mintlify.com/docs/editor/branching-and-publishing#preview-your-changes).
