The Mintlify Editor is a browser-based WYSIWYG documentation editor that lets your whole team write, edit, and publish beautiful docs — no code required. Switch between visual and source editing modes, collaborate with teammates in real time using live cursors, and watch your changes render in an instant live preview. When you're ready to ship, the editor keeps Git in sync automatically, supports branching and pull requests directly from the browser, and includes a built-in AI assistant to help you write, reorganize, and configure your docs faster.
