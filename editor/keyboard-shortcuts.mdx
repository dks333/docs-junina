The Mintlify Editor supports keyboard shortcuts for common actions across editor-wide navigation, visual mode formatting, and source mode editing. The shortcuts follow macOS and Windows conventions.

## Editor-wide shortcuts

These shortcuts work regardless of which mode you are in.

| Command | macOS | Windows |
|---|---|---|
| Search files and content | `Cmd+K` | `Ctrl+K` |
| Toggle AI assistant | `Cmd+I` | `Ctrl+I` |
| Switch visual / source mode | `Cmd+Shift+M` | `Ctrl+Shift+M` |
| Toggle diff view | `Cmd+Shift+D` | `Ctrl+Shift+D` |
| Switch Navigation / Files tree | `Cmd+Shift+F` | `Ctrl+Shift+F` |
| Hide / show sidebar | `Cmd+\` | `Ctrl+\` |

## Source mode shortcuts

Source mode uses the Monaco editor (the same editor that powers VS Code), so standard VS Code shortcuts apply.

| Command | macOS | Windows |
|---|---|---|
| Find | `Cmd+F` | `Ctrl+F` |
| Toggle line comment | `Cmd+/` | `Ctrl+/` |
| Indent | `Tab` | `Tab` |
| Outdent | `Shift+Tab` | `Shift+Tab` |
| Move line up | `Option+↑` | `Alt+↑` |
| Move line down | `Option+↓` | `Alt+↓` |
| Duplicate line below | `Shift+Option+↓` | `Shift+Alt+↓` |
| Add cursor at click | `Option+Click` | `Alt+Click` |
| Add cursor above | `Cmd+Option+↑` | `Ctrl+Alt+↑` |
| Add cursor below | `Cmd+Option+↓` | `Ctrl+Alt+↓` |

## Smart substitutions

When typing in visual mode outside of a code block, the editor automatically replaces certain character sequences with their Unicode equivalents.

| You type | Result |
|---|---|
| `->` | → |
| `<-` | ← |
| `=>` | ⇒ |
| `<=` | ≤ |
| `>=` | ≥ |
| `!=` | ≠ |
